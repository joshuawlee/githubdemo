Fingerprints

A secure HTTPS connection provides users privacy as it prevents other users from obtaining personal information, such as your credit card information and passwords. Furthermore, it provides secure communication between the user and remote web server (Gibson Research Corporation, 2020). This was all possible due to encryption from a technology that is unbreakable (Gibson Research Corporation, 2020). The development of this new enhancement of user security and privacy came at a price in the early days of the Internet, so it was only used for sensitive information. However as technology advanced and the cost of this new connection became more affordable, many websites switched from HTTP to HTTPS. This transition led to the Going Dark Problem where organizations and schools were forced into searching for new ways to monitor and filter information. In response to this issue, they developed a new technology known as HTTPS Proxy Appliances. This technology is a device that allows for the organizations and schools to be inbetween a user and their application, allowing for a way through the unbreakable security and privacy of HTTPS connections. Similarly, there is MITM (man-in-the-middle). MITM presents risks to a man in the middle attack, which is when a perpetrator positions himself in a conversation between a user and an application in order to eavesdrop or impersonate the user.
